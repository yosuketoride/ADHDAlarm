# 忘れん坊アラーム 実装状況まとめ
_最終更新: 2026-03-19_

---

## アーキテクチャ概要

```
RootView
├── OnboardingContainerView（初回のみ）
│   ├── HookView（掴み）
│   ├── MagicDemoView（デモ体験）
│   ├── PermissionsCTAView（権限要求）
│   └── WidgetGuideView（ウィジェット設置ガイド）
└── MainTabView（メイン）
    ├── Tab 0: VoiceInputTab（音声入力）
    ├── Tab 1: AlarmListTab（予定リスト）
    └── Tab 2: SettingsTab → SettingsView（設定）
```

**技術スタック**
- iOS 26+ / AlarmKit / EventKit / AVFoundation / Speech
- SwiftUI + @Observable / async-await / StoreKit 2 / WidgetKit
